# 🏦 Personal Finance Management System

## 📌 Introduction
This is a console-based C++ application built as an assignment for my **Object-Oriented Programming (OOP)** course. The goal of this project was to step away from basic scripts and apply core OOP concepts to simulate a simple financial management system.

## 🛠️ Tech Stack & Concepts
* **Language:** C++
* **Core Concepts Applied:** 
    * **Inheritance & Polymorphism:** Creating a base class for generic accounts, and derived classes for specific types.
    * **Encapsulation:** Keeping sensitive data (like account balances) private and using getter/setter methods securely.

## 🚀 Key Features
* **Multiple Account Types:** Supports basic logic for **Bank**, **Stock**, and **Cryptocurrency** accounts.
* **User Roles:** Simple authorization to differentiate between *Regular* and *VIP* users.
* **Basic Validations:** Prevents invalid actions, such as withdrawing more money than the current balance.

## 🧠 What I Learned
This assignment was a practical exercise for me to deeply understand and apply two core OOP principles: **Encapsulation** and **Inheritance**. I gained hands-on experience in hiding sensitive data (like account balances) from outside interference, and building a base parent class to share common attributes with derived classes, which significantly reduced code duplication.
